НКО Карта - Система управления некоммерческими организациями
О проекте
Веб-приложение для визуализации и управления данными некоммерческих организаций на интерактивной карте России. Система предоставляет возможности поиска, фильтрации и добавления новых организаций.

Архитектура проекта

  Frontend
  HTML5 + Tailwind CSS + JavaScript

  Yandex Maps API для отображения карты

  Адаптивный дизайн с современным UI

Backend
  Python FastAPI - RESTful API

  MySQL - реляционная база данных

  SQLAlchemy - ORM для работы с БД
